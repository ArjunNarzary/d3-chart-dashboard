# Chart Dashboard using D3.js

This is a dashboard that will display charts using d3.js

## Technologies used

- ReactJS
- NodeJS
- ExpressJS
- Typescript
- MaterialUI
- D3.js
- Tailwindcss

## Getting Started

Run client server:

```bash
cd client
# and
npm install
# and
npm run dev
```

Run backend server:

```bash
cd server
# and
npm install
# and
npm run dev
```
